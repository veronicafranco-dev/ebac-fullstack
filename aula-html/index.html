Segundo subtítulo, usando h3

Como você pode ver, esse subtítulo ficou menor que o primeiro, pois existe uma sequência de tags h1..h6, onde respectivamente h1 é maior e h6 é o menor.

Vamos ver de forma exemplificada para entender melhor:

tag h2

tag h3

tag h4

tag h5
tag h6

Como você pôde ver, a tag 'h1' não foi exemplificada aqui, pois ela é a única que não deve ser usada mais de uma vez no mesmo código, como boas práticas. Isso se dá porque em qualquer conteúdo/página, só há um TÍTULO PRINCIPAL, não é mesmo?

Além disso, é extremamente importante para a semântica do conteúdo, pois a tag h1 indica ao navegador, motores de busca e aos leitores qual é o tema principal do conteúdo abordado.

Para outras sessões de destaque, usamos os outros 'h's', como subtítulos, podendo ser utilizados um dentro do outro (indentados), isso ajuda a definir uma estrutura hierárquica ao nosso conteúdo.
